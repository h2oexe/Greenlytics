Bu klasör frontend tarafından doğrudan servis edilen görsel dosyaları içerir.

Şu anda uygulamanın beklediği dosyalar:

- `login-forest.jpg`
  Login ekranındaki büyük arka plan görseli.

- `dashboard-spotlight.png`
  Dashboard spotlight kartının arka plan görseli.

## Önerilen Özellikler

### `login-forest.jpg`

- En az `1600x1200`
- Sisli, doğal, gerçekçi orman / çam teması
- Ortada çok yoğun detay yerine kenarlarda nefes alan kompozisyon
- Auth kutusunun üstüne rahat oturacağı yumuşak kontrast

### `dashboard-spotlight.png`

- En az `1600x900`
- Yatay kompozisyon
- Koyu yeşil filtreyle birlikte de okunabilir duracak görsel
- Metin alanının geleceği orta-sol bölgede aşırı karmaşa olmaması iyi olur

## Kullanım

Dosyaları bu klasöre doğru adla koyman yeterli.

Ardından:

- `npm run dev` açıksa sayfayı yenile
- değilse frontend’i yeniden başlat
